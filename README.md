# prueba-Flask-Bautista-Moyano

## Entorno Virtual

Para crear el entorno virtual:

```
python -m venv .venv
```

Para activar el entorno virtual:

```
source .venv/bin/activate
```

Para instalar las dependencias:

```
pip install flask
```

## Como correr el programa

Solo en la maquina local:

```
flask run
```
